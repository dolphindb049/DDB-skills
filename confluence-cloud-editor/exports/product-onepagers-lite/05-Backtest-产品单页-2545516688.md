### 产品概述
**一句话定义：**

DolphinDB 策略回测引擎是一个基于 DolphinDB 数据库的库内量化策略测试与评估平台，提供从数据存储、因子计算、策略回测到模拟仿真的一站式解决方案。

**核心定位：**
它是 DolphinDB 一站式量化投研平台的核心组件，旨在解决量化投资领域策略研发、测试、优化周期长、精度低、实盘与回测结果偏差大等核心痛点。它连接了投研与开发，实现了“产研代码一体化”。

**目标用户：**
量化研究员、策略分析师、算法交易工程师、做市商、券商与私募 IT 开发/架构师、金融科技公司的风控与交易系统开发人员。

**私募多因子策略选股量化策略团队**
**客户背景**： 原使用**米筐 Python 回测框架**，投研团队持续扩张。需完成**从单因子开发 → 多因子选股 → 策略回测 → 因子绩效归因**的全流程量化投研工作，且希望实现团队协作效率提升、研究成果可比性强。

- 团队扩张后，**缺乏统一的因子管理平台**，投研人员“重复造轮子”（各自开发因子、策略，精力浪费）。

- 不同投研人员的工作**缺乏统一基准对比**，难以快速评估策略/因子的优劣，迭代效率低。

基于 **DolphinDB Starfish**​ 完成全流程覆盖：

- 工具链整合：单因子开发 → 多因子组合优化选股 → 多因子选股策略回测 → 策略结果可视化 → 因子绩效归因。

- 平台化管理：通过统一平台减少重复开发，支持团队在同一基准下对比工作成果。

- **效率提升**：减少重复开发，“一个同事跟踪迭代”即可支撑团队需求，投研效率显著优化。

- **可比性增强**：所有投研工作在同一基准下对比，策略/因子迭代、评估更便捷，加速投研成果落地。

- **全流程覆盖**：从因子到归因的端到端支持，避免环节割裂，缩短策略从研究到实盘的验证周期（虽未给具体数字，但隐含“时间成本大幅降低”）。

**客户背景**： 期货公司面向 **C端个人投资者**（如散户、小型资管客户），需为其提供**量化策略回测、模拟交易、实盘交易**的一体化服务，满足客户“策略研发→回测验证→模拟→实盘”的全流程需求。C端客户自主开发/导入期货量化策略。通过期货公司平台完成**历史行情回测**（验证策略有效性）、**模拟交易演练**（熟悉规则、优化参数），最终对接**实盘交易**（下单成交、数据记录）。

- **数据门槛高**：C端客户获取高质量历史/实时行情数据（含因子、tick级数据）困难，且数据格式不统一，增加策略开发难度。

- **回测效率低**：传统回测工具（如Python本地脚本）处理大规模数据（如全市场tick级）时，计算速度慢、资源占用高，策略迭代周期长。

- **模拟与实盘脱节**：模拟交易与实盘交易的撮合规则、手续费、滑点等细节不一致，导致“回测表现好，实盘亏损”的偏差，客户信任度低。

- **策略管理分散**：客户策略版本混乱、多策略并行管理困难，团队协作（如投顾与客户、客户内部）缺乏统一平台。

- **数据层**：

历史数据：提供**全市场行情+因子数据**，支持“回测/实时订阅”，适配不同策略的数据需求。

- 实时数据：通过“实时数据订阅”通道，将实时行情推送给策略，支持**盘中实时训练/回测**。

- **策略层**：

离线回测策略：支持“历史回测+实时订阅”双模式，策略脚本（如DolphinDB Script/Python）可复用，训练数据格式统一。

- 盘中实时训练策略：基于实时数据订阅，策略可**在线训练、生成信号**，适应盘中动态市场环境。

- **回测引擎层**：

创建回测引擎：支持“历史行情数据表（可回放）+ 策略信号表/模型”注入，模拟真实交易环境（撮合、滑点、手续费、保证金等）。

- **交易层**：

模拟交易：生成“交易表+收益表”，记录模拟成交明细、收益曲线，支持可视化分析。

- 实盘交易：实时更新“收益表+订单表”，通过**MQ消息发布**推送给C端客户，客户端（如VNPy/CTP）订阅后下单，成交数据写入“交易数据库”。

- **数据入库**：回测结果、交易数据分别写入“回测数据库”“交易数据库”，实现数据沉淀与追溯。

- **数据获取效率**：C端客户无需自行爬取/清洗数据，平台提供标准化数据接口，**数据准备时间从“天级”缩短至“分钟级”**。

- **回测性能提升**：基于DolphinDB的分布式计算，**全市场tick级策略回测时间从“小时级”压缩至“分钟级”**（对比本地Python回测，性能提升10-50倍）。

- **策略管理便捷**：统一平台支持多策略版本管理、回测结果对比，**策略迭代周期从“周级”缩短至“小时级”**。

- **客户体验优化**：模拟交易+实盘交易无缝衔接，数据可视化（收益曲线、订单明细），**客户留存率提升30%+**，期货公司增值服务收入增长。
