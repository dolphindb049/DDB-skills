# Quality Gates（必须全部通过）

## Gate 1: 因子定义完整

- [ ] 有明确金融逻辑。
- [ ] 有可执行数学表达式。
- [ ] 所有变量在数据源中可映射。

## Gate 2: 计算可执行

- [ ] `.dos` 在最小样本可运行成功。
- [ ] `.dos` 在全样本可运行成功。
- [ ] 结果无全空、无全常数。

## Gate 3: 入库可复现

- [ ] 统一因子表 schema 正确。
- [ ] 同键唯一（trade_date, securityid, factor_name）。
- [ ] version 与 source_report 已写入。

## Gate 4: 评价有效

- [ ] 可产出 IC 序列与 ICIR。
- [ ] 可产出分层净值与多空净值。
- [ ] 指标可导出并被 Python 读取。

## Gate 5: 报告可交付

- [ ] 报告包含图表与结论。
- [ ] 输出目录可追踪（按因子名+时间戳）。
- [ ] 新 AI 可按文档独立复现。
